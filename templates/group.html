ГРУППОВЫЕ ЗАНЯТИЯ

К Вашим услугам просторный зал аэробики. В клубе представлено более 15 групповых программ с использованием специального оборудования и без него. Кроме того, в каждом направлении есть занятия как для начинающих, так и для гурманов аэробики. Классы для начинающих очень бережно вводят вас в тренировочный процесс. В нашем Клубе каждый может найти себе занятие по душе и выбрать из большого количества предложенного то, что соответствует темпераменту, уровню подготовленности, желанию или просто, вашему настроению.

Действуют и специальные группы для детей от 4 до 12 лет, где опытные педагоги помогают малышам развивать таланты и способности.

{% set img1 = "/static/img/group1.jpg" %}

Йога

Тренировка, состоящая из специальных упражнений, выполняющихся в определенном порядке. Укрепляет и растягивает мышцы, сухожилия и связки, способствует достижению душевного и физического равновесия. Исправляет осанку. Развивает гибкость. Для всех уровней подготовки.

{% set img1 = "/static/img/group2.jpg" %}

Cycle

Групповые занятия на велотренажёрах. Для всех уровней подготовки.

{% set img1 = "/static/img/group3.jpg" %}

Кроссфит

Cпортивное движение, основанное на философии многостороннего физического развития. Эксперты определяют кроссфит как систему подготовки, развивающую все физические качества атлета – выносливость, силу, скорость, гибкость, координацию.

{% set img1 = "/static/img/group4.jpg" %}

Fitball

Силовой урок с использованием мячей. Укрепляет как поверхностные, так и глубокие мышцы тела, снимает нагрузку с позвоночника, развивает чувство равновесия. Особенно показан людям с варикозным расширением вен. Для всех уровней подготовки.

{% set img1 = "/static/img/group5.jpg" %}

Pump

Супер тренировка с мини-штангой. Всем кто хочет поддержать форму, улучшить фигуру, избавиться от лишних кг! Прорабатываются основные группы мышц. Для всех уровней подготовки.

{% set img1 = "/static/img/group6.jpg" %}

Стретчинг

Тренировка, состоящая из специальных упражнений, позволяющих вернуть, либо развить вашу гибкость. Тренировки полезны и необходимы всем независимо от возраста и степени развития гибкости. Для всех уровней подготовки.

{% set img1 = "/static/img/group7.jpg" %}

Step

Аэробный урок с использованием степ платформ, где аэробная нагрузка средней и высокой интенсивности чередуется силой. Для начинающих.

{% set img1 = "/static/img/group8.jpg" %}

ММА

Боевые искусства, представляющие собой сочетание множества техник, школ и направлений единоборств. ММА является полноконтактным боем с применением ударной техники и борьбы как в стойке (клинч), так и на полу (партер).

{% set img1 = "/static/img/group9.jpg" %}

Бокс

Тренировки включают в себя отработку техники нанесения ударов и защиты от них, изучение боевых комбинаций, спарринги, упражнения в парах и на снарядах.
